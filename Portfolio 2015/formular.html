Mit JavaScript Formulare Prüfen

Wenn man Formulare im Internet anbieten, kann man jeden Unsinn in das Formular schreiben und es dann abschicken. Betroffen davon sind vor allem Eingabefelder. Dadurch erhält man z.B überflüssige E-Mails, die die Eingaben automatisch verarbeitenund speichern. Mit Hilfe von JavaScript kann man Formularfelder überprüfen, bevor das sie abgeschickt werden. Bei fehlenden Eingaben kann man das Absenden des Formulars verhindern.
